環境の構築（Github）

Github

GithubはGitというバージョン管理システムをウェブ上で利用できるサービスです。 ここではGithubをバージョン管理のためには使わず、専らウェブサイトを公開するためだけに使うこととします。

Github Desktopのインストール

ここではGithubを簡単に使うためのソフトウェア「Github Desktop」をインストールします。 まず公式サイトにアクセスしてください。 これをダウンロード、インストールします。

インストール後，ログイン画面になるので次のID,Passwordを入力してログインしてください。

User ID : guidematerials
Password : 2019tasukuyasui

セットアップを完了して，次の画面になったらいったん止まってください。

Googleドライブの同期の停止

競合を防ぐため，Google Driveの同期を一旦停止します。

タスクバーからGoogle Driveのメニューを開き，右上のメニューボタンから「一時停止」を選択します。

フォルダの中身を削除

次に同期のためのフォルダの中身を一旦削除します。 エクスプローラーでGoogleドライブフォルダを開いたのち，「Googleドライブ > guide-materials > guide-materials」フォルダを開きます。二つ目のguide-materialsフォルダを使うので間違えないように注意してください。

フォルダに入ったら，いったんそのフォルダ内のファイル・フォルダをすべて削除します。

Github Desktopのセットアップ

先ほどのGithub Desktopの「Let’s get started」画面を開きます。 開いたら，「WRESS/guide-materials」を選択したのち，「Clone WRESS/guide-materials」ボタンを押してクローンします。

このような画面になるので，「choose」ボタンを押して，同期するフォルダを変更します。 先ほどフォルダの中身を削除した空の「guide-materials内のguite-materials」フォルダを選択して，「clone」ボタンを押します。

これによりGithub上のファイルがクローンされます。

ここまで終わったら，先ほどと同様の操作でGoogleドライブの同期を再開してください。

以上でGithub Desktopのセットアップが終わりました。
